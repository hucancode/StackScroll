# Compile
- Android Studio 4
- graddle 5.1.1
- [APK](https://github.com/hucancode/StackScroll/releases)

# Control
- Drag: Scroll through cards.
- Tap: Open a card.
- Long press: Reoder cards.
- Double tap: Add a card.
